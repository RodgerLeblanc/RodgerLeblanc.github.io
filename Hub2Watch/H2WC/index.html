Hub2Watch Color Configuration

Use this configuration page to choose the settings you would like applied to this app.
Colors
Choose the app's top background color.
Choose the app's bottom background color.
Choose the time and date color. The icons color will always remain black, with some white filling once in a while, mostly for weather icons (clouds).
Number Of Slots
How many slots to use. If you set this field to 0 or leave it empty, it will auto-switch between 1, 4 and 6 slots depending on how many unread accounts there is to show. This option will take effect next time H2WC updates its unread count.
